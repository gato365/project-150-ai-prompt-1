## BULK EMAIL SEND NOTES

When sending Email 2 (code delivery), use a **mail merge** approach:

### Option A — Canvas Inbox (recommended)
1. Go to Canvas → Inbox → Compose
2. Select individual students one at a time (or use a course message with individual mode)
3. Paste the template and fill in `[First Name]` and `[CODE]`

### Option B — R mail merge with `gmailr`
Use the `email_list.csv` generated by `assign_groups.R`:

```r
library(gmailr)
library(tidyverse)

email_list <- read_csv("data/email_list.csv")

gm_auth_configure(path = "gmail_credentials.json")

email_list |>
  pwalk(function(name, email, code, group, ...) {
    first_name <- str_extract(name, "^\\S+")  # or reverse: "Last, First"

    body <- glue::glue("
Hi {first_name},

Your participant code for Tuesday's in-class activity is:

**{code}**

Write this down or keep this email open on your phone during class...
[rest of template]
    ")

    gm_mime() |>
      gm_to(email) |>
      gm_subject("Your STAT 252 Experiment Code — Needed for Tuesday 4/29") |>
      gm_text_body(body) |>
      gm_send_message()

    Sys.sleep(1)  # rate limit
  })
```

> ⚠️ Test with your own email address first before bulk send.
